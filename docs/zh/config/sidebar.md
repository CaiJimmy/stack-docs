# 侧边栏

与左侧侧边栏相关的设置。

字段位于 `[Params.sidebar]` 部分。

## compact

- 类型：`bool`
- 默认值：`false`

启用侧边栏的紧凑版本。

## emoji

- 类型：`string`

显示在头像上方的 Emoji。

## subtitle

- 类型：`string`

显示在站点标题下方的副标题。

## avatar

- 类型：`string`
- 默认值：`img/avatar.png`

头像图片的路径或 URL。如果使用本地文件，它应放置在您的 `assets/` 目录中（例如，`assets/img/avatar.png`）。
