# 主题支持的 FrontMatter 字段

| 字段        | 介绍                                                           | 默认值                           |
| :---------- | :------------------------------------------------------------- | :------------------------------- |
| description | 文章简介                                                       |                                  |
| image       | 特色图片                                                       |                                  |
| comments    | 显示 / 隐藏评论区                                              | `true`                           |
| license     | 文章协议 输入 `false` 可以隐藏                                 | `params.article.license.default` |
| hidden      | 隐藏文章（不在首页，归档等页面显示，但是可以直接通过链接访问） | `false`                          |
| math        | 加载 KaTeX 脚本                                                |                                  |
| toc         | 显示 / 隐藏目录                                                | `params.article.toc`             |
