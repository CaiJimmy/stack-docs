# 欢迎

Stack 是一个为博客设计的简单卡片式 Hugo 主题。以下是一些特性：

* 响应式图片支持
* 图片懒加载
* 深色模式
* 本地搜索
* [PhotoSwipe](https://photoswipe.com/) 集成
* 归档页面模板
* 全原生 JavaScript，不使用 jQuery 或任何其他框架
* 无 CSS 框架，保持简单和极简
* 适当裁剪的缩略图
* 子章节支持
* 文章目录
