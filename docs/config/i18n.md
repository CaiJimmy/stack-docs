# i18n

Hugo has built-in support for multilingual sites. You can find more information about it in the [official documentation](https://gohugo.io/content-management/multilingual/).

Translation files are placed in the `i18n` directory. The file name is the language code. For example, the `en.toml` file is the translation file for English.

In order to use a language, set `DefaultContentLanguage` to the language code in the configuration file. For example, if you want to use English, set `DefaultContentLanguage` to `en`.

Currently, the theme supports the following languages:

* `ar`: Arabic
* `be`: Belarusian
* `bg`: Bulgarian
* `bn`: Bengali
* `ca`: Catalan
* `cs`: Czech
* `de`: German
* `el`: Greek
* `en`: English
* `es`: Spanish
* `fa`: Persian
* `fr`: French
* `hi`: Hindi
* `hu`: Hungarian
* `id`: Indonesian
* `it`: Italian
* `ja`: Japanese
* `ko`: Korean
* `nl`: Dutch
* `oc`: Occitan
* `pl`: Polish
* `pt-br`: Portuguese (Brazil)
* `pt-pt`: Portuguese (Portugal)
* `ru`: Russian
* `sk`: Slovak
* `th`: Thai
* `tr`: Turkish
* `uk`: Ukrainian
* `vi`: Vietnamese
* `zh`: Chinese (Simplified)
* `zh-hant-hk`: Chinese (Traditional) (Hong Kong)
* `zh-hant-tw`: Chinese (Traditional) (Taiwan)

> This list might be outdated. Please check the [`i18n` folder in the theme repository](https://github.com/CaiJimmy/hugo-theme-stack/tree/master/i18n) for the most up-to-date and complete list of supported languages.
